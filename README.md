# TraductorIA - Tema: Contabilidad

Este repositorio contiene un archivo SQL que crea una base de datos local en **PostgreSQL** llamada `db_contabilidad`.  
La cual genera una tabla con palabras clave relacionadas con el tema **Contabilidad**.

---

## Contenido

- **db_Contabilidad.sql**  
  Contiene:
  - Crear la base de datos `db_contabilidad`
  - Crear la tabla `palabras_contabilidad`
  - Inserta 15 palabras relacionadas con el tema de contabilidad, junto con su porcentaje de identidad y sinonimos.
